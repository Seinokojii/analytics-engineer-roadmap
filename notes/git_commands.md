# Мои заметки по Git

## Основные команды

### git status
Показывает состояние репозитория (что изменено, что готово к коммиту)

### git add <файл>
Добавляет файл в "staging area" (подготовка к коммиту)
- git add . — добавить все изменённые файлы

### git commit -m "сообщение"
Создаёт "снимок" изменений с описанием

### git push
Отправляет коммиты на GitHub

### git pull
Скачивает изменения с GitHub

### git branch
Показывает список веток
- git branch feature-name — создать новую ветку

### git checkout <ветка>
Переключиться на другую ветку
- git checkout -b feature-name — создать и переключиться

### git merge <ветка>
Слить изменения из другой ветки в текущую
